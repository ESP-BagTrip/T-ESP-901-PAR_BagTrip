╔════════════════════════════════════════════════════════════════════════════╗
║ 🏗️ ARCHITECTURE                                                           ║
╚════════════════════════════════════════════════════════════════════════════╝

PATTERN: BLoC (Business Logic Component)

FLOW:
  User Action (Tap Tab)
       ↓
  BottomTabBar.onTabChanged()
       ↓
  NavigationBloc.add(NavigationTabChanged)
       ↓
  NavigationBloc._onTabChanged()
       ↓
  emit(NavigationState)
       ↓
  BlocBuilder rebuilds UI
       ↓
  Correct page displays

DEPENDENCIES
  flutter_bloc: ^9.1.1 (existing)
  bloc: ^9.0.0 (existing)
  equatable: ^2.0.5 (NEW)
